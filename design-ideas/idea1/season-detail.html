Quick Actions

Implementation Notes for Vue 3 + PrimeVue 4

Hero Section: Custom component with CSS gradients and absolute positioning. Use computed() for dynamic status badges.

Stats Bar: Grid layout with grid-cols-5. Values come from Pinia store seasonDriverStore.stats.

Tabs: PrimeVue Tabs, TabList, Tab, and TabPanel components with custom styling via :pt passthrough.

Panels: Use BasePanel and PanelHeader components for consistent styling across all information panels.

Championship Leader: Custom card with gradient background. Data from standings API endpoint.

Recent Results: Clickable rows navigate to round detail. Position badges use bg-gradient-to-br for metallic effect.

Animations: Use onMounted() with staggered delays or Vue's TransitionGroup for entrance animations.

Icons: Phosphor Icons via @phosphor-icons/vue components like <PhTrophy :size="20" />.

All Tailwind classes shown are production-ready. See design-system.md for complete component examples with Vue 3 Composition API and TypeScript.
