Founded in 2022, we've grown to become one of the premier ACC leagues with over 70 active drivers across multiple skill divisions.

Tailwind CSS PrimeVue Phosphor Icons

Data Hierarchy & Design Notes

League → Competitions (F1, F2, F3, Le Mans, Bathurst 1000, GT World Challenge)

Competition → Seasons (F1 2025, F1 2024, Le Mans 2024/25, etc.) - Expandable/collapsible view

Season → Rounds (Round 1, Round 2, etc.) - Lives on the Season Detail page (next to create)

Key Features in This Design:

  • Expandable competition cards showing seasons
  • Active season highlighted with green border and progress bar
  • Completed seasons shown below active (limited to 2, with "View All" modal)
  • Edit/Delete buttons for both competitions and seasons
  • "New Season" button in each competition
  • Season status indicators: Active (green pulse), Upcoming (blue), Completed (gray)
  • All seasons link to Season Detail page
  • Modal for viewing all archived/completed seasons
  • Empty state for competitions with no active season

PrimeVue Components to Use:

  • Accordion - For expandable competition cards
  • Dialog - For archived seasons modal
  • Button - All action buttons
  • ProgressBar - Season progress
  • Tag - Status badges
  • ConfirmDialog - Delete confirmations
