USAGE NOTES

Use --bg-dark for the main background with the grid pattern overlay. Use --bg-panel for headers, sidebars, and navigation. Use --bg-card for content cards and containers. Use --bg-elevated for hover states and nested elements.

SEMANTIC MEANING

Cyan: Primary actions, links, active states. Green: Success, online status, positive changes. Orange: Warnings, pending states. Red: Errors, danger, destructive actions. Purple: Special features, premium content.
