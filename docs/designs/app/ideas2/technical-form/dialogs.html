//DIALOGS

Confirmation and alert dialogs with semantic icons. Use for destructive actions, warnings, success confirmations, and informational messages.

Dialog Types

Delete Driver?

This will permanently remove Max Verstappen from the roster. This action cannot be undone.

Danger / Destructive
For delete confirmations and destructive actions

Unsaved Changes

You have unsaved changes. Are you sure you want to leave? Your changes will be lost.

Warning
For warning about potential data loss or important decisions

Entry Created

The new driver has been successfully added to the roster.

Success
For confirming successful operations

Season Locked

This season has been finalized. Results can no longer be modified.

Information
For informational notices and acknowledgments
DIALOG GUIDELINES

Dialogs are centered in the viewport with a backdrop overlay. Use danger for destructive actions (delete, remove). Use warning for potential data loss. Use success to confirm completed actions. Use info for general notices. Keep messages concise and action-oriented.
