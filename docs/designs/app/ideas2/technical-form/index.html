//DESIGN_PRINCIPLES
Technical Precision

Use monospace typography for data display and technical labels. Consistent spacing based on 8px grid. Sharp 6px border radius for modern feel.

Data-First Design

Optimize for data readability with high contrast text on dark backgrounds. Use subtle grid patterns to create depth without distraction.

Semantic Color

Color conveys meaning: cyan for primary actions, green for success, orange for warnings, red for errors, purple for special states.

Engineering Aesthetic

Embrace the technical nature with //LABELS, terminal-style output, and system status indicators. Form follows function.
