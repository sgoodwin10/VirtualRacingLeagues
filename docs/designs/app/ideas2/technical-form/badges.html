USAGE GUIDELINES

All badges use var(--font-mono) for consistent technical styling. Use semantic colors consistently: .badge-green for success states, .badge-orange for warnings, .badge-red for errors. Position badges P1-P3 use traditional podium colors (gold, silver, bronze).
